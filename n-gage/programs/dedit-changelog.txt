0.76 R5 01/03/2009
--------------
* Сильные перестановки в меню
* Изменен диалог подтверждения замены
* В меню добавлены возможности - Полноэкранный режим и открыть в кодировке - раньше было доступно только по хоткеям
* Ускорено открытие больших файлов
* Информация о различных действиях в программе отныне выводится не в заголовке документа, а в отдельном окошке
+ Открыть новый файл в новом окне
+ Шаблоны можно создавать прямо в программе
+ Цифровая навигация по меню (а-ля JBakTaskman)
+ Новые переменные в шаблонах:
    * $clipboard - текст из буфера;
    * $filename - имя файла с расширением;
    * $filepath - путь к файлу без имени;
    * $filetitle - имя файла без расширения;
    * $fileext - расширение файла, с точкой.
